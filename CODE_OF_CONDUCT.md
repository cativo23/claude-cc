# Code of Conduct

lumira follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The full text is canonical — this file is a pointer plus the project-specific reporting channel. Maintainers and contributors are expected to read and abide by the linked document.

## Reporting

If you experience or witness behavior that violates the Code of Conduct, contact the maintainer:

- Email: **cativo@cativo.dev**

Reports are handled confidentially. The maintainer will acknowledge within 72 hours and investigate. Retaliation against reporters is itself a violation.

Do **not** use the issue tracker, Discussions, or PRs for Code of Conduct reports — those channels are public.

## Scope

This Code of Conduct applies to all project spaces:

- The lumira GitHub repository (issues, PRs, Discussions, code review threads)
- Any official communication channel tied to the project
- Public spaces when an individual is representing the project (e.g., posting on social media as a maintainer)

## Enforcement

Violations result in actions ranging from a private warning to a permanent ban from project spaces, scaled to severity and intent. The maintainer makes the final call.
